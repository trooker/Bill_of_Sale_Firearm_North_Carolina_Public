# Bill_of_Sale_Firearm_North_Carolina_Public
Publicly available hardcopy of a Bill of Sale for a Firearm within North Carolina between a Private Seller and a Private Buyer.

This version should be used by individuals who might be uncomfortable using browser input objects like buttons, text entry objects, email, etc.  The checkboxes, radio buttons, push buttons, dropdown list of selections, signature blocks, hot links, date-time stamps, etc. are not available.  Their functionality must be performed by the Seller and Buyer.  This document should be completed by the Seller and Buyer in the same manner as the FFL Firearms Transaction Record ATF Form 4473 (August 2023) https://www.atf.gov/firearms/docs/4473-part-1-firearms-transaction-record-over-counter-atf-form-53009/download. 

# Dislcaimers:
1. The author is not a lawyer but has reviewed the NC Statues that apply to firearm sales in North Carolina. The author has had experience selling/buying firearms under NC and FFL statutes. The author is a holder of an active NC CCH permit.

2. The bill-of-sale document presented here is an adaptation from https://eforms.com/bill-of-sale/nc/north-carolina-gun-bill-of-sale/ form for North Carolina as posted 250915.

3. A fellow private gun buyer/seller also provided a bill-of-sale document successfully employed in Texas. The origin of this document could not be determined. But it appears to be in wide usage in Texas.

# List for Firearms Categories and Seller Business 
The hardcopy version has two dropdown list from which the Seller should make selections regarding the category of Firearm being transferred and the type of business in which the Seller is engaged.  If Seller's business type selection/entry is "non-applicable " and does not involve FFL (Federal Firearms Licensing) then the Seller may not have a tax obligation to the NC Department of Revenue (See the Disclaimers abvoe).
